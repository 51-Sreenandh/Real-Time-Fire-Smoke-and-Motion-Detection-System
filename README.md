# Smart Monitoring System Using Python and OpenCV

## Overview

A real-time monitoring system that detects:

* Motion
* Fire
* Smoke

using a webcam and OpenCV.

The system provides:

* Live monitoring
* Alarm alerts
* Email notifications
* Snapshot capture
* Alert logging

## Technologies Used

* Python
* OpenCV
* Tkinter
* Pillow
* SMTP
* Threading

## Features

* Motion Detection
* Fire Detection
* Smoke Detection
* Email Alerts
* Snapshot Storage
* Alert History Logging
* GUI Dashboard
